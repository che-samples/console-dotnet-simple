# console-dotnet-simple

# Commands to run

| #       | Description           | Command  |
| :------------- |:-------------| :-----|
| 1      | Update Dependencies | `cd ${current.project.path} && dnu restore` |
| 2      | Run      |   `dnx -p ${current.project.path}/project.json run` |

# Application Output

The app will print current date to the console
